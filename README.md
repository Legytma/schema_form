## Instalation

* Add this to your package's pubspec.yaml file:
```
dependencies:
  schema_form: "^0.1.2"
```
* You can install packages from the command line:
  with Flutter:
```
$ flutter packages get
```

* Import it Now in your Dart code, you can use:
```
 import 'package:schema_form/schema_form.dart'; 
```
